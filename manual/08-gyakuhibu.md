# 逆日歩と逆日歩狩り

逆日歩(ぎゃくひぶ、品貸料)は制度信用取引で株不足が起きたときに発生する調達コスト。**売り方全員が支払い、買い方全員が受け取る**。優待クロスでは「一般信用を使って逆日歩を避ける」のが基本だが、あえて制度信用で賭ける「制度クロス」、受け取る側に回る「逆日歩狩り(逆クロス)」という応用がある。

*調査日: 2026-07-07(最高料率の算定式は 2026-07-15 に日証金の規則原文・早見表PDFで確認)。規制は日証金公式(taisyaku.jp)で最新を確認すること。*

## 逆日歩の仕組み

1. 制度信用の株は**日証金(日本証券金融)**が貸借取引で調達している
2. 銘柄ごとに毎営業日、融資(買い)と貸株(売り)を突き合わせ、**貸株超過(株不足)**になると日証金が**品貸入札**で機関投資家等から株を借りる
3. その調達コスト=**品貸料(逆日歩)**が、制度信用の売り方への請求・買い方への支払いとして転嫁される

| 項目 | 内容 |
|---|---|
| 支払う人 | **制度信用の売り方全員**(株数比例。逃れられない) |
| 受け取る人 | **制度信用の買い方全員**(一般信用の買い方は受け取れない) |
| 決まり方 | 品貸入札: 料率の低い申込みから採用し、必要株数に達した料率がその日の品貸料 |
| **判明タイミング** | **約定翌営業日**(優待クロスなら権利落ち日の昼頃に速報)。**事前には分からない** |

## 逆日歩の計算

```
逆日歩 = 品貸料率(円/株/日) × 株数 × 品貸日数
```

**品貸日数(逆日歩日数)が最大の落とし穴**。新規売建の受渡日から返済受渡日の前日までの**暦日数(土日祝を含む)**で数える:

| 権利付最終日のクロス | 品貸日数 |
|---|---|
| 通常(火〜木曜受渡し) | 1日分 |
| 新規建ての受渡日が金曜(**権利確定日が金曜**のパターン) | **3日分**(土日を挟む) |
| 年末年始・GWなど連休絡み | **4〜7日分**(実例: 2025年12月26日権利付最終日は6日分) |

制度クロスを検討するときは、まず**その権利日の品貸日数が何日か**を確認する。

## 最高料率(逆日歩の上限)と倍率適用

- 品貸入札には上限=**最高料率**があり、次の式で決まる(日証金規則、2026-07-15 に規則原文で確認): **投資単位(貸借値段×売買単位)が5万円以下 → 品貸料の上限100円、5万円超 → 5万円から1万円以下を増すごとに+20円**。1株あたりの最高料率 = 上限÷売買単位(1円以下は1円、1円超は10銭の整数倍に切り上げ)。売買単位100株なら 株価1,000円→2円/株、3,000円→6円/株、5,000円→10円/株(概ね株価の0.2%水準)
- ただし以下の場合、**上限が引き上げられる**:

| 状況 | 倍率 |
|---|---|
| 権利落日の6営業日前〜2営業日前 | 2倍 |
| **権利付最終日(権利落日の前営業日)** | **4倍** |
| 注意喚起銘柄(通知日翌営業日〜取消日前営業日) | 2倍 |
| 申込制限・申込停止措置銘柄(実施日〜解除日前営業日) | 2倍 |
| 異常な貸株超過(臨時措置) | 4倍 |
| 極めて異常な貸株超過(臨時措置) | **10倍** |

倍率は「上限」に適用されるもので、実際の品貸料が自動で倍になるわけではない。それでも**権利付最終日の制度売りは最悪ケースが跳ね上がる**構造になっている。条件が重複した場合(権利付最終日×注意喚起など)は乗算で重ねがけ(4倍×2倍=8倍)とする実務解説が多いが、規則原本では未確認(要確認)。2025年12月末権利では注意喚起・申込停止銘柄が最高料率(通常の2倍)に達した実例がある。

**最悪ケースの試算例**: 株価1,000円・100株、最高料率が仮に2円/株/日 → 権利付最終日は4倍で8円/株/日。品貸日数3日なら 8円×100株×3日 = **2,400円**。優待価値と比べてこの最悪額が許容できるかで判断する。

## 信用規制 — 売り禁・注意喚起・増担保

制度クロス・逆日歩狩りの前に必ず確認する規制。**日証金(貸借取引=制度信用向け)と取引所・証券会社(信用取引全般)の2系統**がある。

### 日証金の規制3段階

| 段階 | 内容 | 影響 |
|---|---|---|
| **貸株注意喚起** | 将来の制限措置の可能性を通知・公表(残高基準や株価急変等で発動) | 最高料率2倍。制度クロスは原則見送り |
| **貸株申込制限** | 1社1日あたりの貸株申込株数を制限 | 最高料率2倍。さらなる悪化を想定 |
| **貸株申込停止措置(俗称: 売り禁)** | 貸株申込み自体を停止 | **制度信用の新規売り+買い方の現引きが不可**。最高料率2倍 |

- 公表場所: **日証金サイト「制限措置等一覧」(通常16:30頃更新)**。各社の取引画面・注意銘柄ページにも反映される
- 相場急変時は注意喚起を経ずにいきなり売り禁になることもある。個別銘柄の理由・期間は非公表
- **売り禁でも既存の売り建玉は維持でき、返済(買い戻し・現渡し)は可能**。止まるのは新規の貸株申込みと買い方の現引きであり、優待クロスの出口(現渡し)は塞がれない
- 一般信用は日証金を経由しないため直接の対象外(ただし後述の社内規制あり)。一般信用買いの現引きも日証金の停止措置の影響を受けない

### 取引所・証券会社の規制

| 規制 | 内容 | 優待クロスへの影響 |
|---|---|---|
| **日々公表銘柄** | 信用残高を毎日公表(それ自体に取引制限なし) | **増担保規制の予備軍**。制度・大口クロスは慎重に |
| **増担保規制** | 委託保証金率の引き上げ: 第一次**50%(うち現金20%)** → 第二次70%(40%)→ 第三次90%(60%)→ 第四次は新規売買禁止 | **新規建てのみ対象で既存建玉への遡及なし**。現金部分は代用有価証券で代替できず資金拘束が急増(制度・一般とも対象) |
| 証券会社の社内規制 | 各社判断による新規建て停止・在庫供給停止など | 「(社内)」表示や売建可能数量で確認 |

### 規制時、一般信用売りはどうなるか(4社)

日証金規制は制度信用が対象だが、各社は**社内規制**として一般信用売りを別途止めることがある:

- **楽天**: 日証金の申込停止(楽天呼称「申告規制」)で制御されるのは制度信用のみ。一般信用は「**一般信用社内規制**」として別途実施され、[信用取引規制銘柄一覧](https://www.rakuten-sec.co.jp/ITS/Companyfile/margin_restriction.html)に実施日・解除予定日つきで表示される
- **SBI**: 当社判断で規制銘柄化・在庫供給停止(「本日の注意銘柄」ページ、毎営業日17時頃更新)。一方で**貸株注意喚起銘柄でも HYPER空売りの対象として扱う場合があると公式明記** — 制度が売り禁でも HYPER で建てられることがある(高コスト・1泊定石は変わらず)
- **eスマート**: 売建可能銘柄一覧から除外されると**発注済みの注文も失効**。調達困難銘柄はプレミアム料付空売りの枠組みへ
- **日興**: 売建可能数量の表示で実質制御。日証金規制との連動明文は未確認(要確認)

### 建玉済みの一般信用売りが強制返済されることはある?

**ある(ただし段階を踏む)**。SBI の公式規定: 在庫不足日の24時頃に繰越の新規売建注文を失効(無期限のみ)→ 解消しなければ**翌営業日の後場寄付前に返済注文を強制執行**。当日20:00頃にメッセージボックスへ通知が来る。無期限建玉は「18:30の在庫不足判定 → 翌営業日後場寄付で強制返済」の場合もある。eスマートは催告期間つきの返済期日変更 → 期日超過で強制決済。**フライングクロス中の銘柄が規制入りしたら放置しない**(対処 → [11-recovery.md](11-recovery.md))。

### 規制サイン → とるべき行動(早見表)

| サイン | 行動 |
|---|---|
| 日々公表銘柄入り | 制度・大口クロスは慎重に。増担保入りを想定した資金計画 |
| 増担保規制 | クロス済みなら影響なし。新規は現金拘束(50%・現金20%〜)を計算してから |
| 貸株注意喚起 | 制度クロス・逆日歩狩りは原則見送り(最高料率2倍)。一般在庫か HYPER の有無を確認 |
| **売り禁(申込停止)** | 制度での新規クロス不可。**既存建玉の現渡しは可能なので慌てない**。一般信用は各社の社内規制表示を確認 |
| 「一般信用新規売停止(社内)」 | 他社の在庫へ回る。建玉済みなら毎晩の通知(SBI は20時のメッセージボックス)を確認 |
| 在庫不足・強制返済の通知 | 放置せず自主的に現渡し/返済して仕切り直し |

### 制度クロス前のチェックリスト(逆日歩リスクの読み)

- **日証金の規制状況**: 上記3段階のどれかに入っていないか
- **応札ランク**(2024年11月から日証金が公表): 品貸入札の逼迫度をA(応札倍率1.0〜1.2倍、最逼迫)〜Fで表示。約定日翌営業日10:40頃更新
- **貸借残(証金残)**: 貸株残が融資残を上回っている(株不足)かどうか
- **過去の逆日歩実績**: 同銘柄の過去権利日の逆日歩を調べる(高逆日歩の常連銘柄がある)
- **品貸日数**: その権利日が連休絡みでないか

## 制度クロスの損得([02-methods.md](02-methods.md) パターンc)

- 期待値: 逆日歩が僅少なら「制度貸株料(年1.10〜1.15%、1〜数日分)だけ」で済み、一般信用より安い
- リスク: 高逆日歩を引くと優待価値を大きく超える損失。**2025年12月のトレードワークスでは1株あたり計48円(6日分相当)の逆日歩となり、3万株で約144万円の負担**(優待価値5万円)という事故例
- 定石: 権利付最終日だけ跨ぐ(日数最小化)、品貸日数が多い権利日は避ける、規制・応札ランク・過去実績を確認、最悪ケース額を計算してから

## 逆日歩狩り(逆クロス)

高逆日歩が予想される銘柄で、**受け取る側=制度信用の買い方**に回る手法。優待は取れない(現物を持たないため)。純粋に逆日歩の受取りを狙う。

**組み方**: 制度信用買い + 一般信用売り の両建て(価格変動をヘッジ。売りを一般信用にするのは自分が逆日歩を払わないため)

1. 権利付最終日までに「制度信用買い」+「一般信用売り」を同数量・寄付前成行で同時発注
2. 権利付最終日の大引けを跨ぐ(建玉のまま。現引きしない — 現引きすると制度買い建玉が消えて逆日歩を受け取れない)
3. 権利落ち日以降に決済(現引き→現渡し、または反対売買で同時解消)

**損益構造**:

| 収入 | 支出 |
|---|---|
| 逆日歩 × 株数 × 品貸日数(制度買い建玉分) | 一般信用売りの貸株料 |
| 配当調整金の受取り(制度買い: 配当の84.685%) | 配当調整金の支払い(一般売り: 配当の**100%**)→ **差額15.315%が持ち出し** |
| | 買方金利、名義書換料(買建玉が権利確定日を跨ぐため1単元55円)、手数料 |

```
利益 = 逆日歩収入 −(貸株料 + 配当×15.315% + 名義書換料 + 金利)
```

- 配当が大きい銘柄ほど持ち出しが増え、必要な逆日歩のハードルが上がる(実例分析: 積水ハウス1,000株・配当45円/株のケースで「100株あたり逆日歩700円超なら勝ち」という試算)
- **逆日歩が0銭〜僅少ならコストだけ確定で負け**。本質的に投機であり、毎年高逆日歩が付く常連銘柄(高配当×優待人気×貸借在庫が細い)を過去実績から選ぶのが定石
- 優待クロス勢が使う一般信用売り在庫を自分も使うため、在庫争奪は同じ土俵

**対になる手法 — 配当クロス**: 現物買い + 制度信用売りで、受取配当(源泉前100%)と支払調整金(84.685%)の**差15.315%を狙い、逆日歩を支払う側**に立つ手法。逆日歩がその差益を食い潰すリスクとの天秤で、同一銘柄で逆日歩狩りとどちらが有利かは逆日歩の予想水準次第。

## 配当落調整金の整理(税務は [09-faq.md](09-faq.md))

| 立場 | 受払 | 配当比 |
|---|---|---|
| 現物買い | 受取 | 100%(源泉徴収20.315%後の手取り79.685%) |
| 制度信用売り | 支払 | **84.685%** |
| 制度信用買い | 受取 | 84.685% |
| 一般信用売り | 支払 | **100%**(主要社共通。全社は要確認) |

配当落調整金は配当所得ではなく**譲渡損益**として扱われる(特定口座なら自動で損益通算)。
