# SMBC日興証券

優待クロス視点の要点: **ダイレクトコースなら信用手数料・現引き・現渡しが完全無料**だが現物手数料は有料。そのため**「一般信用売り+制度信用買い→現引き→現渡し」が定番**(現物買いを経由せず現物株を無料取得)。一般信用売りは**3年型のみ・貸株料 年1.90%**。在庫数量は17:10〜17:15頃に更新され、**注文受付が再開する17:30が争奪戦**(先着順)。

**⚠️ 料率改定に注意**: 信用取引の金利・貸株料は2025年12月・2026年3月に連続改定された(一般貸株料 1.40%→1.70%→**1.90%**)。ネット上の古い記事は旧料率(1.40%等)のままのことが多い。

*調査日: 2026-07-07(料率は「2026年4月現在」の公式値。貸株料の日数=両端入れは 2026-07-15 に公式で確認)。発注前に末尾の出典で最新を確認すること。*

## 手数料体系

優待クロスは**ダイレクトコース一択**(一般信用売建はダイレクトコース限定。総合コースは対面体系で不向き)。

| 取引 | 手数料(ダイレクトコース) |
|---|---|
| **信用取引(制度・一般、新規・返済)** | **0円**(約定代金・建玉残高・期間によらず) |
| **現引き・現渡し** | **0円** |
| 現物株式 | **有料**: 10万円以下137円 / 20万円以下198円 / 30万円以下275円 / 50万円以下440円 / 100万円以下880円 / 200万円以下1,650円 / 300万円以下2,200円 / 500万円以下3,300円 / 1,000万円以下4,950円(税込) |

現物手数料は無料にならない(SBI・楽天のような無料化制度なし)。そのため**現物買いを直接せず「制度信用買い→現引き」で現物を取得する**のが日興の定石(下記)。

## 一般信用売り

| 項目 | 内容 |
|---|---|
| 種類 | **3年型のみ**(短期・日計りなし。返済期限は原則3年) |
| 貸株料 | **年1.90%**(2026年3月30日約定分〜) |
| 売方金利 | 0.00% |
| 対象銘柄 | 当社指定銘柄(公式記載は約1,800銘柄・2023年7月末時点。最新は要確認) |
| 配当落調整金 | **配当金と同額(100%)を支払い**(公式明記) |
| 権利処理等手数料 | **売り建玉は対象外**(かからない) |
| 利用条件 | ダイレクトコース+日興イージートレード信用口座+書面電子交付同意 |

**料率改定の履歴**: 〜2026/1/4: 1.40% → 2026/1/5〜: 1.70% → 2026/3/30〜: **1.90%**。金利情勢により今後も変更されうる。
貸株料・金利の日数は「新規建玉の受渡日〜返済の受渡日」の暦日数(土日祝込み・**両端入れ** = 初日と最終日の両方を算入。公式明記)。最短でも2日分。

3年期限なので数週間前のフライングクロスも期限面では余裕だが、**1.90%は楽天・SBIの無期限(1.10%)やeスマート長期(1.10%)より割高**になった点に注意(旧1.40%時代の「日興=フライング最安」は過去の話)。

制度信用買いは「現引きまでのつなぎ」として使う(当日現引きなら金利は約1日分)。制度信用売りは逆日歩リスクあり(→ [08-gyakuhibu.md](08-gyakuhibu.md))。

## 現引き・現渡し

- 手数料: **どちらも無料**
- 当日扱いの締切: **15:30まで**の注文が当日約定扱い(公式FAQは接続不可のため要確認。実務サイト複数で確認)
- 翌営業日扱いの予約: 大引け後の**受付再開(17:30)以降**に発注可能
- **権利落ち日の現渡しは、権利付最終日の17:30以降に予約発注しておくのが定石**
- 現引・現渡の取消は営業日5:00〜17:30の間は不可(公式明記)

## 優待クロス定番手順とコスト

**定番: 一般信用売り + 制度信用買い → 現引き → 現渡し**

1. 17:30の受付再開で一般信用売り在庫を確保
2. 同じ夜のうちに「一般信用売り(成行)」+「制度信用買い(成行)」を**同時発注** → 翌朝の寄付で同値約定
3. 約定後**当日中(15:30まで)に現引き**(制度信用買い建玉 → 現物株、無料)。買方金利を約1日分に抑え、買い建玉で権利確定日を跨ぐことによる権利処理等手数料(55円/単元)も回避
4. 権利付最終日の大引けを跨ぐ
5. **権利付最終日の17:30以降に現渡しを予約発注**(権利落ち日約定扱い・無料)で解消

コスト内訳(2026年4月現在の料率):

| 項目 | 金額 |
|---|---|
| 信用手数料(売り・買い)/現引き/現渡し | すべて 0円 |
| 一般信用貸株料 | 建玉金額 × **1.90%** × 日数 ÷ 365(受渡日ベース・暦日数・**両端入れ**) |
| 制度信用買方金利 | 建玉金額 × **3.27%** × 1日 ÷ 365(当日現引き = 受渡日が同日でも両端入れで1日分) |
| 現物手数料 | 0円(経由しないため) |

試算例(日数は両端入れのため、最短の1日クロスでも貸株料は2日分):
- **最短の1日クロス(30万円、火曜建て→水曜現渡し)**: 貸株料 30万×1.90%×2日÷365 ≒ 31円 + 買方金利 30万×3.27%×1日÷365 ≒ 27円 = **約58円**(現物買い直接なら手数料440円)
- **100万円を1週間前にクロス(貸株日数8日)**: 貸株料 100万×1.90%×8÷365 ≒ **416円** + 買方金利約90円(現物買い直接なら880円+貸株料)

## 注意点

- **信用取引口座の開設が必須**(審査あり。委託保証金30%以上かつ30万円以上、維持率25%)
- **一般信用売建はダイレクトコース限定**。総合コースでは不可
- 信用手数料無料は日興イージートレード(オンライン)経由の注文のみ
- **管理費(信用管理費)は不要**(公式明記)— 長期のフライングクロスでも管理費はかからない
- **権利処理等手数料55円/単元**は「買い建玉が権利確定日を跨いだ場合」のみ。権利付最終日までに現引きすれば回避。**売り建玉にはかからない**
- 配当銘柄では配当落調整金(配当と同額)の支払いと現物配当(源泉後79.685%)の差額約20.3%が一時発生(→ [09-faq.md](09-faq.md))
- **大引け(15:30)〜17:30は注文受付不可**。現渡し予約・翌日分のクロス発注は17:30を待つ。20:15〜20:20の中断にも注意
- 建玉上限: 同一銘柄原則20億円以内かつ発行済株式数の1%まで。**東証グロース・名証ネクスト銘柄は原則3,000万円以内**
- 整理・監理銘柄、取引所規制銘柄、当社独自判断銘柄は売建不可。空売り価格規制(トリガー抵触時の51単元以上成行制限)も適用
- **料率は改定が続いている**(2025年12月・2026年3月)。クロスのコスト計算前に必ず公式の最新料率を確認
