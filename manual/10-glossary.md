# 用語集

優待クロスの実務で使う用語。50音順ではなく、関連グループごとにまとめている。

*調査日: 2026-07-07。*

## 取引の基本

| 用語 | 意味 |
|---|---|
| **優待クロス(つなぎ売り)** | 買いと信用売りを同時に建てて価格変動リスクを消し、優待権利だけを取る手法 |
| **クロス取引** | 同一銘柄・同数量の買いと売りを同時に約定させること |
| **両建て** | 同一銘柄の買いポジションと売りポジションを同時に持つ状態 |
| **現物買い** | 通常の株式購入。株主権利(優待・配当・議決権)を得る |
| **信用買い** | 証券会社から資金を借りて株を買う。買方金利がかかる。優待はもらえない(名義が株主でないため) |
| **信用売り(空売り)** | 証券会社から株を借りて売る。貸株料がかかる |

## 決済方法

| 用語 | 意味 |
|---|---|
| **現渡し(品渡し)** | 保有する現物株を渡して信用売り建玉を返済する方法。優待クロスの出口。手数料は4社とも無料 |
| **現引き(品受け)** | 資金を渡して信用買い建玉を現物株として引き取る方法。「制度信用買い→現引き」で現物手数料を回避するのに使う |
| **返済売買(反対売買)** | 建玉を市場での売買で決済すること。優待クロスでは通常使わない(価格変動リスクが生じるため) |

## 日程

| 用語 | 意味 |
|---|---|
| **権利確定日** | 株主名簿の基準日。多くは月末 |
| **権利付最終日** | 権利確定日の2営業日前。この日の大引けまでに買っていれば権利獲得 |
| **権利落ち日** | 権利付最終日の翌営業日。現渡しはこの日以降(発注予約は権利付最終日の夜から可能な会社が多い) |
| **受渡日** | 約定の2営業日後(T+2)。貸株料・金利の日数計算は受渡日ベース |
| **大引け** | 後場の取引終了。東証は 15:30(2024年11月5日から延長) |
| **寄付(よりつき)** | その日最初の約定。優待クロスは寄付前の成行同時発注が定石 |

## 信用取引の種類

| 用語 | 意味 |
|---|---|
| **制度信用取引** | 取引所ルールの信用取引。返済期限6ヶ月。売りは貸借銘柄のみ。逆日歩リスクあり |
| **一般信用取引** | 証券会社独自ルールの信用取引。逆日歩なし。売り在庫は各社の調達力次第 |
| **貸借銘柄** | 制度信用で売り(貸株)ができる銘柄。日証金が貸株を扱う |
| **日計り信用(デイトレ信用)** | 当日返済が前提の一般信用。楽天「いちにち信用」、SBI「日計り信用/HYPER空売り」、イースマ「デイトレ信用」など |

## コスト関連

| 用語 | 意味 |
|---|---|
| **貸株料** | 信用売りで株を借りるコスト。建玉金額 × 年率 × 日数 ÷ 365(日数は受渡日ベース・両端入れ) |
| **買方金利** | 信用買いのコスト。建玉金額 × 年率 × 日数 ÷ 365(日数は受渡日ベース・両端入れ) |
| **両端入れ(りょうはいれ)** | 金利・貸株料の日数の数え方。新規建ての受渡日と返済の受渡日の**両方を算入**する(=受渡日の差+1日)。4社共通。最短の1日クロスでも貸株料は2日分になる |
| **逆日歩(品貸料)** | 制度信用で株不足のとき、売り方が支払い買い方が受け取る料金。入札で決まり事前に分からない → [08-gyakuhibu.md](08-gyakuhibu.md) |
| **プレミアム料** | 一般信用売りの人気銘柄に上乗せされる料金(イースマの入札制プレミアム料、SBI の HYPER料など) |
| **配当落調整金** | 権利日を跨いだ信用建玉で授受される配当相当額。売り方が支払い(制度84.685%・一般100%)、買い方が受け取る。税務上は譲渡損益 |
| **名義書換料(権利処理手数料)** | 権利確定日を跨いだ信用**買**建玉にかかる費用。1単元55円(ETF・ETNは5.5円)が一般的。売建玉への適用は会社により要確認 |
| **事務管理費(管理費)** | 建玉が1ヶ月経過するごとにかかる費用(1株11銭、最低110円・上限1,100円が目安)。短期のクロスでは通常発生しない |

## 実務・戦術

| 用語 | 意味 |
|---|---|
| **在庫** | 一般信用売りに使える株数の残り。各社が調達した分しかなく、人気銘柄は争奪戦 |
| **在庫開放(在庫補充)** | 証券会社が一般信用売り在庫を追加・更新するタイミング。夕方〜夜が中心で、時刻は会社ごとに異なる → [07-comparison.md](07-comparison.md) |
| **フライングクロス** | 権利日よりかなり早く(数週間前など)クロスして在庫を確実に確保する戦略。貸株料が日数分かさむ |
| **直前クロス** | 権利付最終日の直前(前日夜〜当日朝)にクロスしてコストを最小化する戦略。在庫が残っていないリスクあり |
| **制度クロス** | 制度信用売りでのクロス。在庫の心配はないが逆日歩が青天井 |
| **逆日歩狩り(逆クロス)** | 高逆日歩を見込んで受け取る側(制度信用買い+一般信用売り)に回る手法 → [08-gyakuhibu.md](08-gyakuhibu.md) |
| **配当クロス** | 現物買い+制度信用売りで、受取配当と支払調整金の差(配当の15.315%)を狙う手法。逆日歩を支払う側 → [08-gyakuhibu.md](08-gyakuhibu.md) |
| **空クロス** | 優待権利月以外の基準日にクロスし、株主名簿への記載回数を稼ぐ継続保有条件対策 → [09-faq.md](09-faq.md) |
| **片約定(片玉)** | 買いと売りの一方だけ約定してしまった状態。価格変動リスクに晒される。優待クロス最大の事故 |
| **寄成(よりなり)** | 寄付での約定を狙う成行注文(執行条件)。クロスの同時約定に使う |
| **SOR** | 複数市場から最良価格を探す注文ルーティング。楽天ゼロコース・eスマート手数料無料化の適用条件になっている |
| **継続保有条件(長期保有条件)** | 「1年以上保有」など、権利日だけの保有では優待がもらえない条件 |
| **端株(単元未満株)保有** | 継続保有条件対策として1株だけ持ち続け、株主番号を維持するテクニック(有効性は企業の判定方法次第) |
| **一般売り残(在庫)検索** | 各社サイト・アプリで一般信用売りの可能数量を確認する機能 |

## 制度・機関

| 用語 | 意味 |
|---|---|
| **日証金(日本証券金融)** | 制度信用の株・資金を証券会社に供給する証券金融会社。逆日歩の入札を実施 |
| **証金残(貸借残)** | 日証金が公表する融資残・貸株残。株不足(貸株超過)だと逆日歩の可能性 |
| **品貸日数(逆日歩日数)** | 逆日歩がかかる日数。受渡日ベースの暦日数で**土日祝を含む**(通常1日、連休絡みで最大7日程度) → [08-gyakuhibu.md](08-gyakuhibu.md) |
| **最高料率** | 品貸入札の上限料率。投資単位(株価×売買単位)で決まり、**権利付最終日は4倍**に引き上げられる |
| **応札ランク** | 日証金が公表(2024年11月〜)する品貸入札の逼迫度指標。A(最逼迫)〜F |
| **貸株注意喚起・申込制限・申込停止** | 日証金による貸借取引の規制3段階。段階が進むほど株不足=逆日歩リスク大。対象銘柄は最高料率も2倍に → [08-gyakuhibu.md](08-gyakuhibu.md) |
| **売り禁(うりきん)** | 貸株申込停止措置の俗称。制度信用の新規売りと買い方の現引きが不可になる。**既存建玉の維持・現渡しは可能** |
| **増担保規制(ましたんぽ)** | 取引所による委託保証金率の引き上げ(第一次50%・うち現金20%〜、段階的に強化)。**新規建てのみ対象**で既存建玉に遡及しない。制度・一般とも対象 |
| **日々公表銘柄** | 信用残高が毎日公表される銘柄。それ自体に制限はないが**増担保規制の予備軍** |
| **T+2** | 株式の受渡しが約定日から2営業日後であること(2019年7月16日から) |
