# 手法パターン

優待クロスの手法は「売り側を何で建てるか」「買い側を何で建てるか」の組み合わせで決まる。売り側の主流は逆日歩のない一般信用売り。買い側は、その証券会社の**現物手数料が無料かどうか**で最適解が変わる。

*調査日: 2026-07-07(貸株料の日数=両端入れは 2026-07-15 に4社公式で確認)。料率は各社ページ([03](03-nikko.md)/[04](04-rakuten.md)/[05](05-sbi.md)/[06](06-esmart.md))を参照。*

## 手法の選び方(早見)

| 手法 | 売り | 買い | 向くケース |
|---|---|---|---|
| **a. 現物クロス** | 一般信用売り | 現物買い | 現物手数料が無料の会社(楽天ゼロコース、SBIゼロ革命、eスマートSOR)。**初心者はまずこれ** |
| **b. 現引きクロス** | 一般信用売り | 制度信用買い→**現引き** | 現物手数料が有料の会社(**SMBC日興**)。資金効率を上げたい場合 |
| **c. 制度クロス** | 制度信用売り | 現物買い | 一般在庫が枯れた銘柄をどうしても取りたい上級者。**逆日歩リスクあり** |
| **d. 変種** | — | — | いちにち信用買い→現引き、フライング/直前クロス、空クロスなど |

いずれも出口は共通: **権利落ち日に現渡し**(保有現物で売り建玉を返済。4社とも手数料無料)。

## a. 一般信用売り + 現物買い → 現渡し(基本形)

1. 一般信用売りの**在庫を確保**(各社の在庫開放時間: → [07-comparison.md](07-comparison.md))
2. 権利付最終日までに、**寄付前(市場が閉まっている時間帯)に「現物買い」と「一般信用売り」を同数量・成行で同時発注**
3. 翌朝の寄付で同値約定 → 権利付最終日の大引けを跨ぐ
4. 権利落ち日に現渡し

**コスト** = 売買手数料(無料の会社なら0)+ 貸株料 + 配当銘柄なら配当調整差額(→ [09-faq.md](09-faq.md))

```
貸株料 = 建玉金額 × 年率(1.10〜3.90%) × 日数 ÷ 365
※日数は新規建て受渡日〜返済受渡日の暦日数・両端入れ(初日と最終日の両方を数える。4社共通)。
  最短(火曜建て→水曜現渡し等の平日連続)でも2日分。土日祝を挟むと増える
```

最もシンプルで事故が少ない。逆日歩は構造的に発生しない。

## b. 一般信用売り + 制度信用買い → 現引き → 現渡し

現物買いを「無料の信用買い+無料の現引き」に置き換えて、**現物手数料を回避する**手法。

1. 在庫確保後、寄付前に「制度信用買い」と「一般信用売り」を同数量・成行で同時発注(信用同士のクロス)
2. 寄付で同値約定 → **当日中に現引き**(品受け)して買い建玉を現物化
3. 権利付最終日の大引けを跨ぎ、権利落ち日に現渡し

**なぜ安くなるのか**(3点セット):
- 信用取引の手数料を現物より安く(無料に)設定している会社が多い
- 現引き・現渡しの手数料は無料
- 当日中に現引きすれば買方金利は最小の**1日分**(金利の日数も両端入れのため、現引きの受渡日が建玉受渡日と同日でも1日分かかる)

**現引きを当日(遅くとも権利付最終日まで)に済ませる理由**:
1. **優待の権利は現物保有者のみ**。信用買い建玉のままでは優待が付かない(現引き忘れは致命的事故)
2. 買建玉のまま権利確定日を跨ぐと**名義書換料**(1単元55円)がかかる
3. 買方金利(年2.8〜3.3%)が日数分かさむ

**コスト** = 貸株料 + 買方金利(最小日数)+ 配当調整差額。SMBC日興(信用手数料0円・現物有料)での定番(→ [03-nikko.md](03-nikko.md))。

注意: 買い側は制度信用銘柄であること。楽天・SBI・eスマートは現物買いが無料化されているため、**手数料目的でこの手法を使う意味は薄い**(信用買いは保証金約3割で建てられるため、資金効率目的では今も有効)。

## c. 制度信用売り + 現物買い → 現渡し(制度クロス)

一般信用の在庫制約がない代わりに、**逆日歩(品貸料)が青天井**のギャンブル性を持つ手法。貸借銘柄のみ可能。

1. **権利付最終日に**(早く建てるほど逆日歩リスク日数が増えるため直前が定石)寄付前に「現物買い」+「制度信用売り」を同時発注
2. 権利落ち日に現渡し

**コスト** = 貸株料(制度は年1.10〜1.15%と安い)+ **逆日歩(額は権利落ち日まで不明)** + 配当調整差額(制度売りは84.685%で一般売りより小さい)

- 逆日歩が僅少なら一般信用より安上がり。ただし高逆日歩を引くと優待価値を大きく超える損失例がある(2025年12月のトレードワークス: 1株あたり計48円(6日分相当)の逆日歩、3万株で約144万円負担。優待価値5万円)
- 権利付最終日は品貸の最高料率が**4倍**に引き上げられる。土日祝・連休を挟むと日数も増える
- 仕組み・リスク管理・過去実績の調べ方 → [08-gyakuhibu.md](08-gyakuhibu.md)

## d. 変種・応用

**いちにち信用買い → 現引き**(買いコストの圧縮)

日計り商品(楽天「いちにち信用」、SBI「日計り信用」、eスマート「デイトレ信用」)は手数料無料・金利0%。これで買い建てて**当日中に現引き**すると、現物取得コストがほぼ0になる。手数料有料コースの利用者や1日定額の枠を残したい場合に有効。
- **現引き忘れは翌営業日強制決済+手数料(楽天・eスマートとも2,200円)**
- 日計りの**売り**は当日返済必須のため、権利跨ぎの売り玉には使えない(売りは一般信用の無期限/短期/長期で)

**フライングクロス**(早取り)

権利日の数週間〜1ヶ月以上前、在庫があるうちにクロスする。人気銘柄を確実に確保できる代わりに貸株料が日数分かさむ。
- 期限のない低率商品(楽天無期限1.10%、SBI無期限1.10%、eスマート長期1.10%、日興3年型1.90%)が向く。楽天短期(14日)・SBI短期(15営業日)は期限があるので不可
- SBIは優待銘柄が原則「短期」区分のため無期限で売れないことが多く、短期の通常開始は「新規建可能日(権利付最終日を含めて14営業日目)の前夜19:00」。さらに1営業日早い裏技「**フライングクロス**」(返済期限=権利付最終日とし、権利落ち日の期限超過強制決済で決済する。**現渡し不可**)もある(例: 2026年8月末権利 → 通常注文 8/6(木)19:00〜・フライング注文 8/5(水)19:00〜。→ [05-sbi.md](05-sbi.md))
- 建玉が1ヶ月を超えると事務管理費(1株11銭/月、最低110円等)も発生
- 建玉中に銘柄が規制入り・在庫不足になると**強制返済されることがある**(SBI: 20時通知→翌営業日後場寄付執行など)。フライング中は通知を毎晩確認(→ [08-gyakuhibu.md](08-gyakuhibu.md) の規制節)

**直前クロス**(コスト最小)

権利付最終日の前営業日夜〜当日朝にクロス。貸株料は最小(通常、受渡日ベースで数日分)だが、在庫が残っていないリスクが高い。在庫が豊富な日興や、不人気優待で有効。

**空クロス**(継続保有対策)

優待権利月以外の基準日(中間の名簿判定日)にもクロスして、株主名簿への記載回数を稼ぐテクニック。「同一株主番号で連続N回記載」条件への対策。端株の常時保有と組み合わせる(→ [09-faq.md](09-faq.md))。

**配当クロス / 逆日歩狩り(逆クロス)**

優待ではなく配当調整金の差額や逆日歩そのものを狙う手法 → [08-gyakuhibu.md](08-gyakuhibu.md)

## 同時約定の実務(全手法共通)

大原則: **自分の買いと売りは「板寄せ」で同値約定させる**。板寄せは1日に3回ある — 寄付(9:00)、後場寄り(12:30)、大引け(15:30)。ザラ場中に両建てを直接ぶつけると約定価格がずれるうえ、**仮装売買(不公正取引)と見なされるおそれ**がある。

### 時間帯別 — いつ発注して、どの板寄せで約定させるか

| いま何時? | 発注のしかた | 約定タイミング |
|---|---|---|
| **前日15:30〜当日8:59**(基本形) | 成行で買い・売りを連続発注 | 当日の**寄付(9:00)** |
| **前場中(9:00〜11:30)**に在庫を取れた | すぐ発注せず**昼休みまで待つ**(場中に成行を出すと即約定してバラける) | 後場寄り(12:30) |
| **昼休み(11:31〜12:29)** | 成行で同時発注(eスマート公式もこの方法を案内。SBIは11:30〜12:05が注文停止のためその後に) | **後場寄り(12:30)** |
| **後場中(12:30〜15:30)**に在庫を取れた | **執行条件「引成(引け成行)」**で買い・売りを同時発注 — 対応: **楽天○・SBI○**/**eスマート×(引け発注不可・公式明記)**/日興は要確認 | **大引け(15:30)** |
| 15:30以降 | 夜間に成行で同時発注 | 翌営業日の寄付 |

- **権利付最終日の後場に在庫を確保できた場合、引けクロスがその権利の最後のチャンス**(楽天・SBIのみ。eスマートでは間に合わないので翌回へ)
- 引けクロスの注意: クロージング・オークション導入(2024年11月)で引けの約定機会は増えたが、**値が付かなければ両方失効**(その権利日は見送り)。ストップ配分など特殊な板では数量ずれもありうるため、約定後に両建ての数量を必ず確認
- 楽天はクロスになる注文に執行条件「寄付」「引け」の指定を必須化している(2020年10月31日〜)。ザラ場中の成行同時発注はできない仕様
- ザラ場中に数秒差で成行を入れて組む方法は、スリッページと仮装売買リスクの両方を負うため**本マニュアルでは非推奨**

### 共通の注意

- **ストップ高/安が予想される日は要注意**: 寄らないと片約定(通常、不利な側だけ約定)のリスク。発注取消も検討
- **空売り価格規制**: トリガー抵触銘柄では51単元以上の信用新規売り成行が失効対象(SBI)。規制回避目的の分割発注は取引制限対象。大口クロスは規制条件を事前確認
- 発注後は必ず**注文一覧で銘柄・数量・売買区分・信用区分(一般/制度)を突合**する
