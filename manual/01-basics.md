# 優待クロスの基礎

*調査日: 2026-07-07(貸株料の日数=両端入れは 2026-07-15 に4社公式で確認)。制度(T+2、立会時間など)は安定しているが、変更時は要更新。*

## 優待クロスとは

株主優待の権利だけを、株価変動リスクを負わずに取得する手法。**同一銘柄・同数量の「買い」と「信用売り」を同時に建てる**(= クロス取引、つなぎ売り)ことで、株価が上がっても下がっても損益が相殺される。権利付最終日を跨いで株主名簿に載り、優待の権利を得たあと、権利落ち日に**現渡し(品渡し)**で両建てを解消する。

- 買い側:株主として名簿に載り、優待(と配当)の権利を得る
- 売り側:株価下落(権利落ち)のリスクをヘッジする
- 利益 = 優待の価値 − クロスにかかったコスト(貸株料・手数料・配当調整の差額など)

## 日程のルール

株主優待・配当の権利は「権利確定日」(多くは月末、20日・15日等の銘柄もある)の株主名簿で決まる。株式の受渡しは約定日から **2営業日目(T+2)** なので、逆算して:

| 用語 | 定義 |
|---|---|
| **権利確定日** | 株主名簿に記載される基準日。多くは月末 |
| **権利付最終日** | 権利確定日の **2営業日前**。この日の大引け時点で買い保有していれば権利獲得 |
| **権利落ち日** | 権利付最終日の翌営業日。**この日以降に現渡しすれば権利は確保済み** |

例:権利確定日が 7/31(金)なら、権利付最終日は 7/29(水)、権利落ち日は 7/30(木)。

- 権利確定日が土日祝の場合は直前の営業日が基準日となる(銘柄の定款による)。
- 権利落ち日の現渡しは、**多くの証券会社で権利付最終日の夜(大引け後)から発注予約が可能**。具体的な受付時間は各社ページ参照。

## 基本の流れ(最小構成)

1. **在庫確保・建玉**(権利付最終日まで、当日を含む):一般信用売りの在庫を確保し、同数量の買いを同時に建てる。約定価格を揃えるため**寄付前に成行で同時発注**するのが定石(→ [02-methods.md](02-methods.md))
2. **権利付最終日の大引けを跨ぐ**:買いポジションで株主権利が確定
3. **権利落ち日に現渡し**:保有現物を信用売り建玉の返済に充当。売買手数料なしで両建てが同時に消える(現渡し手数料は4社とも無料)

## コスト構造

優待クロスのコストは以下の合計。**優待価値がコスト合計を上回る銘柄・タイミングだけ仕掛ける**のが大原則。

| コスト | 発生する取引 | 概要 |
|---|---|---|
| **貸株料** | 信用売り | 建玉金額 × 年率 × 日数 ÷ 365。日数は新規建て受渡日〜返済受渡日の暦日数を**両端入れ**(初日・最終日の両方を算入。4社共通)で数える。最短でも2日分、**土日祝を挟むとさらに増える** |
| **売買手数料** | 買い・売りの新規建て | 4社とも条件次第で無料〜低額(各社ページ参照) |
| **買方金利** | 制度信用買い(現引きまでの間) | 建玉金額 × 年率 × 日数 ÷ 365。現引きを早めれば最小化できる |
| **配当調整の差額** | 配当がある銘柄 | 現物側で受け取る配当(源泉徴収後 79.685%)と、信用売り側で支払う配当落調整金の差額。詳細 → [08-gyakuhibu.md](08-gyakuhibu.md) の税務の節と [09-faq.md](09-faq.md) |
| **逆日歩(品貸料)** | 制度信用売りのみ | 株不足時に売り方が支払う。**一般信用売りなら発生しない**。→ [08-gyakuhibu.md](08-gyakuhibu.md) |
| **プレミアム料等** | 一部の証券会社・銘柄 | イースマの「プレミアム料」、SBI の HYPER料など、人気銘柄の一般信用売りに上乗せされる料金 |
| **名義書換料・事務管理費** | 信用建玉の権利跨ぎ・長期化 | 買建玉が権利確定日を跨ぐと1単元55円(名義書換料)。建玉1ヶ月経過ごとに管理費。当日現引き・1ヶ月以内の決済なら概ね回避できる |

## 一般信用と制度信用

| | 一般信用取引 | 制度信用取引 |
|---|---|---|
| 株の調達元 | 証券会社が自社で調達 | 証券金融会社(日証金)経由 |
| 逆日歩 | **なし** | 株不足時に発生(青天井リスク) |
| 売り在庫 | 各社の在庫限り。**人気銘柄は争奪戦** | 貸借銘柄なら原則いつでも売れる |
| 返済期限 | 会社・商品による(無期限/短期/当日) | 6ヶ月 |
| 貸株料 | やや高め(商品による) | 低め(年1.10%前後) |

- 優待クロスの主流は**逆日歩リスクのない一般信用売り**。
- あえて制度信用売りで逆日歩を賭けに行く「制度クロス」、逆日歩を受け取る側に回る「逆日歩狩り(逆クロス)」もある → [08-gyakuhibu.md](08-gyakuhibu.md)

## 在庫争奪と「いつクロスするか」

- 一般信用売りの在庫は有限。人気優待銘柄は**権利日の数週間〜1ヶ月以上前に在庫が枯れる**ことも。
- 早くクロスするほど確保は確実だが、**貸株料が日数分かさむ**(フライングクロス)。直前を狙えばコストは最小だが在庫リスクが高い。銘柄の人気度とコストの天秤。
- 各社の**在庫補充・開放時間**(夕方〜夜が中心)に張り付いて確保するのが実務の要。時刻は各社ページと [07-comparison.md](07-comparison.md) を参照。

## 優待側の注意(クロスで取れない優待)

- **継続保有条件**(「1年以上保有」「同一株主番号で連続 N 回記載」等)がある銘柄は、権利日だけのクロスでは対象外になることがある。端株(単元未満株)の常時保有で株主番号を維持するテクニックが知られているが、企業側の判定方法変更で無効になるリスクあり(→ [09-faq.md](09-faq.md))。
- 優待の内容・条件は変更・廃止されうる。発表(適時開示)を確認してから仕掛ける。
