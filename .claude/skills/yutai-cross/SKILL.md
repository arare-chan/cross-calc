# 優待クロス質問応答

このリポジトリの `manual/` は優待クロス(つなぎ売り)の手引書。質問には**必ず該当ファイルを読んでから**回答する。記憶で数値(料率・手数料・時刻)を答えてはいけない。

- 4社横断の貸株料概算は `manual/07-comparison.md` の試算表を優先して引用する。個別銘柄・特殊条件(プレミアム料、制度金利、配当など)が絡む場合のみ `manual/02-methods.md` のコスト式に各社ページの料率を当てはめて計算過程を示す。

## 回答のルール

1. **数値には必ず「調査時点」を添える**。マニュアル各ページの調査日を確認し、「2026年7月調査時点」のように明記。料率・時刻は変わりうるので、発注直前は公式サイト(各ページ末尾の出典リンク)での確認を促す。
2. **手法の前提を確認する**。同じ質問でも「一般信用売り+現物買い」と「一般信用売り+制度信用買い→現引き」で答えが変わる場合、両方提示するか、どちらの前提か確認する。
3. **コスト試算は式を示す**。貸株料 = 建玉金額 × 年率 × 日数 ÷ 365 のように、代入した値が分かる形で。日数は受渡日ベースで数える(土日祝を挟むと増える)。
4. **リスクの注記**。制度信用売りが絡む回答では逆日歩リスクに必ず触れる。投資判断は自己責任である旨は、しつこくない範囲で。
5. マニュアルに載っていないことを聞かれたら、その旨を明言した上で一般知識として答え、必要なら Web で最新情報を確認する(WebSearch)。マニュアルの改善余地があれば提案する。

## 「マニュアルを最新化して」と頼まれたら

運用方針: 更新は自動ではなく、ユーザーが権利月の前などに手動で依頼する(2026-07 に合意)。依頼されたら以下のフローで実行する:

1. **並列リサーチ**: 4社(日興・楽天・SBI・eスマート)それぞれについて、公式サイトで確認 — ①手数料体系と無料化の条件 ②一般信用売りの料率・返済期限 ③制度信用金利 ④在庫開放・注文受付・現渡しの時間 ⑤「金利・貸株料変更のお知らせ」等の告知ページ ⑥クロス関連の新サービス。各社の告知一覧ページを必ず見る(例: 日興は2026年に2回改定しており、告知ページに履歴がある)
2. **突合**: 現行マニュアル(03〜06)の数値と比較し、変わった箇所だけを更新。出典URL・ページ末尾の調査日も更新
3. **横断反映**: 料率・時刻が変わったら 07-comparison.md(比較表・試算表・タイムライン)、02-methods.md(対応表・料率言及)、09-faq.md の試算にも波及させる。旧数値の取り残しを grep で確認
4. **報告**: 変更点の新旧一覧をユーザーに報告する。変更がなければ「変更なし、調査日のみ更新」と報告

## その他のマニュアル更新ルール

- 数値の更新は出典 URL を確認してから書き換え、ページ末尾の調査日を更新する。
- 新しい証券会社の追加は既存の各社ページの章立て(手数料体系/一般信用売り/制度信用/現引き・現渡し/在庫開放・注文受付時間/定番手順とコスト/注意点/出典)に合わせる。
- スライド版(slides/yutai-cross-slides.html)に料率・時刻を記載している箇所(4社カード・タイムライン・コスト例)も、大きな改定時は忘れずに更新する。
