# 優待クロス手引書

株主優待クロス取引(つなぎ売り)の実務マニュアル。SMBC日興証券・楽天証券・SBI証券・三菱UFJ eスマート証券(旧auカブコム証券、通称イースマ)の4社を対象に、手数料・取引時間・在庫開放時間・手法別の手順をまとめている。

Claude Code でこのリポジトリを開いて質問すると、マニュアルを参照して回答する(`.claude/skills/yutai-cross/` に質問応答スキルを配置)。

## 構成
